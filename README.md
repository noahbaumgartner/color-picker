# ColorPicker

## How to start
Prerequisites:
* Install Node.js
* Install Angular-CLI

Follow the following three steps to start the application:
1. Clone Code
2. Run NPM INSTALL
3. Run NG SERVE

## Functionality
The Color-Picker can be used to pick a hex color. For that there are three sliders:
1. red
2. green
3. blue

So the application is following the RGB-Model in background.
